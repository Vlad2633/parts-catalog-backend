# Parts Catalog  Backend

Серверна частина курсового проєкту **«Каталог автозапчастин»**.

Backend реалізує REST API для роботи з користувачами, категоріями, брендами,
запчастинами та замовленнями.

---

## Технології
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (авторизація)
- ES Modules

---

##  Основний функціонал
- Реєстрація та авторизація користувачів
- Ролі користувачів (USER / ADMIN)
- CRUD для:
  - категорій
  - брендів
  - запчастин
- Оформлення замовлень
- Пошук, фільтрація, пагінація
- REST API для frontend-додатку

---

##  Запуск проєкту

### 1. Встановити залежності
```bash
npm install
```
### 2. Створити файл .env
```
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/parts_catalog
JWT_SECRET=your_secret_key
```
### 3. Запустити сервер
```
npm run dev

Backend буде доступний за адресою:
 http://localhost:5000
